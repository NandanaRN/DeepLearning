# Deep Learning Lab Experiments

A structured collection of laboratory experiments focusing on core Deep Learning architectures and applications, spanning tabular data prediction, computer vision, and natural language processing (NLP).

##  Repository Structure

* **`Lab Experiment 1.ipynb`**: Student Performance Prediction using MLP
* **`Lab Experiment 2.ipynb`**: Waste Classification using CNN
* **`Lab Experiment 3.ipynb`**: News Article Categorization using Simple RNN
* **`student-mat.csv`**: Dataset utilized for the Lab 1 predictive model

---

##  Experiment Overview

###  Lab 1: Student Performance Prediction (MLP)
* **Objective:** Predict student academic performance based on demographic and academic attributes.
* **Architecture:** Multi-Layer Perceptron (MLP)
* **Dataset:** `student-mat.csv`
* **Key Focus:** Tabular data preprocessing, feature engineering, and regression/classification modeling.

###  Lab 2: Waste Classification (CNN)
* **Objective:** Classify waste items into recyclable categories to automate sorting and segregation.
* **Architecture:** Convolutional Neural Network (CNN)
* **Dataset:** [Garbage Classification Dataset (Kaggle)](https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification)
* **Key Focus:** Computer Vision, image classification, and environmental AI applications.

###  Lab 3: News Article Categorization (Simple RNN)
* **Objective:** Classify incoming news articles into predefined thematic categories.
* **Architecture:** Simple Recurrent Neural Network (RNN)
* **Dataset:** AG News Dataset (Hugging Face)
* **Key Focus:** Natural Language Processing (NLP), text classification, and sequential data processing.
